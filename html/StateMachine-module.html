Package StateMachine

source code

This package implements State Machines.

Import module sm to define state machines programatically, or module decorator to define state machines declaratively. It is safe to import * from each of these modules.



Author: Michel Beaudouin-Lafon

Contact: mbl@lri.fr

Version: 0.1

Date: 4 December 2007

Submodules [hide private]
  • StateMachine.decorator: This module extends regular state machines (module sm) with decorators to facilitate the declaration of state machines.
  • StateMachine.sm: This module implements state machines for event-driven programming.
  • StateMachine.smTest: This module tests state machines.
